CMARF - Chief Minister Accident Relief Portal

Role: Java Developer

Description:

The CMARF portal is an online platform designed to streamline the application and approval process for accident relief funds provided by the Tamil Nadu government.

Key Features:

  • Developed backend APIs using Spring Boot to handle form submissions, approvals, and status tracking.
  • Managed MySQL database design and queries for storing applicant details and approval workflows.
  • Created dynamic front-end forms using HTML, CSS, JavaScript for public applications.
  • Implemented SFTP (Secure File Transfer Protocol) integration to send requests and receive fund disbursement responses from Indian Bank.
  • Developed PDF and Excel report generation for exporting applicant data and approval history through Spring Boot.
  • Performed API testing using Postman to validate all backend services, ensuring smooth communication between the front end and database.
  • Worked on user authentication, admin dashboards, and role-based access controls for secure application management.

StartupTN - TANFUND

Role: Java Developer

Description:

The StartupTN portal is a government initiative designed to support and register startups across Tamil Nadu, enabling them to connect with schemes, mentors, and investors.

Key Features:

  • Integrated the Zoom API to allow startups and mentors to schedule, manage, and track online meetings directly from the portal.
    • Created secure meeting tokens.
    • Scheduled meetings, updated, and deleted them via Zoom API.
    • Stored meeting logs and details in the database.
  • Implemented email notification templates using HTML and Thymeleaf for sending meeting invites and updates.
    • Automatically triggered email notifications with meeting links and details.
    • Used dynamic HTML templates for personalized, well-designed emails.
  • Applied data validation and error handling to ensure smooth form submissions and robust API communication.
  • Optimized backend APIs for performance, security, and scalability.
